0.9.1 (07/17/2018)
New Features
  • Post button added to save dialog that will immediately queue the submission to post.
  • Warning notification added for when a description is too long for Twitter (when twitter is the selected option tab).
  • All posts successful or otherwise will now store a log of the post in the logs tab on the submission popup thing.
  • HR in descriptions now supported.
Bug Fixes
  • Fixed a scenario where custom description options would sometimes not appear.
  • Fixed a good amount of strange behaviors with descriptions created using the text editor (notify me with any strange behavior in the future).
  • Derpibooru should detect username more accurately now.
  • Fixed weird issue with duplicate tags being incorrectly removed.
  • Attempted to fix corrupted templates issue (your templates will be lost in this update sorry :( ).
Miscellaneous
  • Derpibooru and e621 no longer get the PostyBirb ad (requested by e621 admins).
  • OAuth officially correctly done (I think).
0.9.0 (07/09/2018)
New Features
  • New Tumblr Additional option to disable/enable adding in the title.
  • Pixiv now supports Additional File uploads.
  • Summary dialog now also shows additional files (no thumbnail yet).
  • Completely changed how posting, queueing, and editing galleries work.

Bug Fixes
  • Websites that are unauthorized should notify that they are unauthorized when posting.
  • Fixed issue with tags not being deletable (tackled 1/2 in 0.8.9).
  • Fixed a stupid dialog typo.
Miscellaneous
  • Updated translations for new stuff.
  • Updated translation files on PostyBirb website.
  • A new line was added to the basic advertise message for PostyBirb.
  • Updated Electron and Electron packager.
