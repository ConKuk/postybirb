0.9.9 (10/16/2018)
New Features
  • Patreon submissions can now be scheduled through additional options. (requested feature)
  • App will now remember the language you set it to.
  • Patreon will now accept pdf, rtf, txt, and md files.
  • Complete remake of PostyBirb's UI. Tutorial disabled for now. All feedback is welcome.
  • App will now attempt to re-upload a Fur Affinity submission to increase resolution limits.
  • Added template manager that lets you delete, view, and rename templates.
  • Can now disable/enable hardware acceleration in settings dialog.
  • Tag field will now block certain characters that most sites consider invalid ";|\\[]=*^%$!`~.?+
Bug Fixes
  • Furry Network now refreshes its authentication keys correctly.
  • App will now wait a few seconds before trying to post a scheduled submission if it opened from tray.
  • Attempt to fix Inkbunny login issues for people with special characters in their password.
  • Fixed issue where Furry Network detecting an invalid tag would cause the post to hang indefinitely.
Miscellaneous
  • Text Editor will no longer force to plain text.
  • Text editor should try to remove inline images (they don't really work).
  • Removed old settings button found in sidebar. Option to clean all scheduled or unscheduled submissions can now be found in settings dialog.
  • Template name minimum length reduced to 2.
  • Big code refactor.
