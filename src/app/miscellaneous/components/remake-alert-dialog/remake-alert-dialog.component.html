PostyBirb 2.0!

The 1.X version of PostyBirb is no longer actively supported.

v1.2.7 adds a setting that allows you to opt out of update prompts.



You can find more about v2 here PostyBirb 2.0
If you want to download and try 2.0 you can download it Here
You will need to manually install v2.

Be aware that templates and unposted submissions will not transfer to v2. Although if you decide to downgrade to v1 you should still be able to use whatever templates you already had.
